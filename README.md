# AI Mental Health Companion 

An AI-powered wellness companion that provides empathetic conversational support, mood tracking, and voice interaction using OpenAI and FastAPI.

 Features

* AI emotional support chatbot
* Mood tracking dashboard
* Voice input (speech-to-text)
* AI voice replies
* Modern calming UI
* Real-time AI responses

 Tech Stack

* FastAPI
* OpenAI API
* HTML/CSS/JavaScript
* Web Speech API

 Future Improvements

* Advanced emotion detection
* Mood analytics charts
* Personalized memory
* Mobile responsiveness
* Dark mode
* AI journaling
